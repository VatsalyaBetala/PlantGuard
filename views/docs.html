Quick start

Get a local PlantGuard instance running and make your first prediction in under two minutes.

1
Clone the repo

git clone https://github.com/VatsalyaBetala/plant-disease-classification.git

2
Install dependencies

From the project root: pip install -r requirements.txt. A Python 3.9+ venv is recommended.

3
Start the server

python main.py — the UI opens at http://127.0.0.1:8000. Model weights auto-download on first run.

4
Analyse an image

Visit /upload-images, drop a leaf photo, and watch the classifier + Grad-CAM heatmap in the Library tab.
